# PRD: Pang Game — Mission 1

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 아케이드 액션 |
| 원작 | Pang (Mitchell Corporation, 1989) |
| 배경 | 일본 후지산 (Mt. Fuji) |
| 플레이어 수 | 1~2인 |

Pang은 화면을 가득 채운 풍선(버블)을 모두 제거하면 스테이지를 클리어하는 아케이드 게임이다.
풍선은 맞을수록 더 작은 풍선으로 분열되며, 가장 작은 단계에서 소멸한다.

---

## 2. 핵심 게임플레이

### 공(버블) 동작
- 공은 **포물선 궤적**으로 튀어 오르며 좌우 벽과 바닥에서 반사된다.
- 크기는 **Large → Medium → Small** 3단계이며, 각 단계에서 피격 시 2개로 분열된다.
- Small 공은 피격 시 완전히 소멸한다.
- 크기가 클수록 튀어 오르는 높이가 높고, 이동 속도는 느리다.

### 플레이어
- 좌우로만 이동할 수 있으며, 점프는 불가능하다.
- 기본 무기는 **작살(Harpoon)**: 위 방향으로 발사하며 천장에 닿을 때까지 선이 유지된다.
- 공이나 공의 파편에 닿으면 즉시 사망한다.

### 무기 시스템
- **Harpoon**: 기본 무기. 수직으로 선을 쏴 공을 제거.
- **Wire**: 발사 후 빠르게 위로 뻗었다 즉시 사라지는 근접형.
- **Double Harpoon**: 좌우 양방향으로 동시에 발사.
- 무기 아이템은 공 제거 시 확률적으로 드롭된다.

---

## 3. Mission 1 구성

| 항목 | 내용 |
|------|------|
| 배경 | 후지산 / 평지형 단순 지형 |
| 초기 공 구성 | Large 공 2개 |
| 총 처리해야 할 공 수 | Large 2 → Medium 4 → Small 8, 총 14개 |
| 난이도 | 입문 (공 이동 속도 느림, 장애물 없음) |
| 제한 시간 | 60초 |

### 스테이지 특징
- 지형은 평평한 단일 바닥으로 구성되어 발판이나 장애물이 없다.
- Large 공 2개가 좌우에서 서로 다른 타이밍으로 튀어오르며 시작한다.
- 입문 스테이지로 공의 이동 속도와 튀어 오르는 높이가 낮아 공 궤적 파악이 용이하다.

---

## 4. 승리 / 실패 조건

| 조건 | 내용 |
|------|------|
| 클리어 | 화면의 모든 공을 제한 시간 내에 소멸 |
| 실패 (사망) | 공 또는 적과 충돌 |
| 타임오버 | 제한 시간 초과 시 목숨 차감 후 재시작 |
| 게임 오버 | 목숨(Life) 소진 |

---

## 5. 점수 시스템

- 공 제거 시 크기에 따라 점수 획득:
  - Large: 낮은 점수
  - Medium: 중간 점수
  - Small: 높은 점수
- 한 번에 연속으로 여러 공을 제거할수록 **콤보 배수** 적용.
- 스테이지 클리어 시 남은 시간에 비례한 **타임 보너스** 지급.
- 아이템 획득 시 추가 점수 부여.
