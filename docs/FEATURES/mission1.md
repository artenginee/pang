# Mission 1

## 스테이지 정보

| 항목 | 내용 |
|------|------|
| 배경 | 일본 후지산 (Mt. Fuji) |
| 지형 | 평지 (장애물 없음) |
| 제한 시간 | 60초 |
| 난이도 | ★☆☆☆☆ (입문) |

---

## 초기 공 구성

| 공 크기 | 초기 수량 |
|---------|-----------|
| Large | 2개 |
| Medium | 0개 |
| Small | 0개 |

### 분열 후 총 처리 수

```
Large 2개
  └─ Medium 4개
       └─ Small 8개  (총 소멸 대상: 8개)
```

Large 2 → Medium 4 → Small 8, 총 **14번** 피격 필요.

---

## 난이도 특성

- 공 이동 속도: **느림**
- 공 튀어오르는 높이: **높음** (Large 기준, 예측 용이)
- 장애물: **없음**
- 지형 발판: **없음** (단순 평지)
- 아이템 드롭률: **보통**

### 초보자를 위한 포인트
1. Large 공은 느리게 포물선을 그리므로 궤적 예측이 쉽다.
2. 공이 분열될 때 좌우로 퍼지므로, 분열 직후 중앙을 피하는 것이 유리하다.
3. Small 공은 빠르고 낮게 움직이므로 분열이 많아질수록 주의가 필요하다.

---

## 진행 흐름

```
스테이지 시작
  → Large 공 2개 화면 좌우에서 등장
  → 플레이어: Harpoon으로 공 제거
  → Large 피격 → Medium 2개 분열
  → Medium 피격 → Small 2개 분열
  → Small 전부 소멸
  → 스테이지 클리어
```

---

## 클리어 조건

- 화면의 모든 Small 공까지 제거 (총 8개 소멸).
- 제한 시간 60초 이내 달성 시 타임 보너스 지급.

## 실패 조건

- 공과 직접 충돌 → 목숨 차감, 스테이지 처음부터 재시작.
- 60초 이내 클리어 실패 → 목숨 차감, 재시작.
