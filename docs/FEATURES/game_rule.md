# 게임 룰 (Game Rules)

## 기본 목표

화면에 등장하는 모든 공(버블)을 제한 시간 내에 작살(Harpoon)로 제거하면 스테이지 클리어.

---

## 플레이어

- 플레이어는 화면 하단에서 **좌우로만 이동**한다. 점프 불가.
- 공이나 공과의 충돌 시 **즉시 사망**, 목숨(Life) 1 차감.
- 기본 보유 목숨: **3개**.
- 목숨이 모두 소진되면 **게임 오버**.

### 조작키

| 키 | 동작 |
|----|------|
| `←` | 왼쪽 이동 |
| `→` | 오른쪽 이동 |
| `Space` | 작살 발사 |

---

## 공(버블) 동작 규칙

### 크기 및 분열
공은 3가지 크기로 존재하며, 피격 시 아래 단계로 분열한다.

```
Large  →  Medium × 2
Medium →  Small  × 2
Small  →  소멸
```

### 이동 패턴
- 공은 **포물선 궤적**으로 튀어 오르며 좌우 벽과 바닥에서 반사된다.
- 크기가 클수록 **높이 높게, 느리게** 이동.
- 크기가 작을수록 **낮고 빠르게** 이동.

| 크기 | 최대 높이 | 이동 속도 |
|------|-----------|-----------|
| Large | 높음 | 느림 |
| Medium | 중간 | 보통 |
| Small | 낮음 | 빠름 |

---

## 무기 시스템

### 기본 무기: Harpoon
- `Space` 키로 수직 위 방향으로 선을 발사.
- 선은 천장 또는 공에 닿을 때까지 유지된다.
- 선이 유지되는 동안 재발사 불가.

### 아이템 무기 (공 제거 시 드롭)
| 무기 | 설명 |
|------|------|
| Wire | 즉시 위로 뻗었다 빠르게 사라지는 근접형. 연사 가능. |
| Double Harpoon | 좌우 양방향으로 동시에 선 발사. |

---

## 아이템

공 제거 시 확률적으로 아이템이 드롭된다. 아이템은 일정 시간 후 사라진다.

| 아이템 | 효과 |
|--------|------|
| 무기 아이템 | Wire / Double Harpoon으로 교체 |
| 시계 | 화면의 모든 공을 일시 정지 |
| 방어막 | 1회 피격 무효화 |

---

## 제한 시간

- 각 스테이지마다 제한 시간이 존재한다.
- 시간 초과 시 목숨 1 차감 후 해당 스테이지 처음부터 재시작.

---

## 점수 시스템

| 상황 | 점수 |
|------|------|
| Small 공 제거 | 고득점 |
| Medium 공 제거 | 중간 |
| Large 공 제거 | 저득점 |
| 연속 제거 (콤보) | 배수 적용 |
| 스테이지 클리어 | 잔여 시간 × 보너스 점수 |
| 아이템 획득 | 추가 점수 |

---

## 승리 / 실패 조건

| 조건 | 결과 |
|------|------|
| 모든 공 제거 | 스테이지 클리어 → 다음 스테이지 진행 |
| 공과 충돌 | 사망 → 목숨 차감 후 재시작 |
| 제한 시간 초과 | 사망 → 목숨 차감 후 재시작 |
| 목숨 0 | 게임 오버 → 메인 화면으로 복귀 |
