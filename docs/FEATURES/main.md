# 메인 화면 구성 (Main Screen)

## 화면 레이아웃

메인 화면은 게임 시작 전 플레이어를 맞이하는 첫 화면이다.
전체 화면을 세로로 구성하며, 중앙 정렬 기반의 단순한 레이아웃을 사용한다.

```
┌─────────────────────────────┐
│                             │
│         PANG                │  ← 게임 타이틀
│                             │
│      [ START GAME ]         │  ← 시작 버튼
│      [ HOW TO PLAY ]        │  ← 조작법 안내
│                             │
│       HI-SCORE: 0           │  ← 최고 점수
│                             │
└─────────────────────────────┘
```

---

## 구성 요소

### 타이틀
- 게임 이름 **"PANG"** 을 상단 중앙에 크게 표시.
- 배경은 단색 또는 간단한 애니메이션(공이 튀는 연출).

### 메뉴 항목
| 항목 | 기능 |
|------|------|
| START GAME | 미션 1부터 게임 시작 |
| HOW TO PLAY | 기본 조작법 및 규칙 안내 화면으로 이동 |

### HI-SCORE
- 현재 세션에서 달성한 최고 점수를 표시.
- 초기값은 0.

---

## 상태 전환

```
메인 화면
   ├── START GAME 선택 → 미션 1 게임 시작
   └── HOW TO PLAY 선택 → 조작법 안내 화면
                              └── 돌아가기 → 메인 화면
```

---

## 조작

| 키 | 동작 |
|----|------|
| `←` / `→` | 메뉴 항목 선택 이동 |
| `Enter` / `Space` | 선택 확정 |
